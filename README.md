[![Release](https://img.shields.io/github/v/release/spider-nest/slim)](https://github.com/spider-nest/slim/releases/latest)

**Static web page generation template**

## Usage

### Development

```
$ pnpm i
$ pnpm dev
```

Git hooks, see the `package.json`

```
pnpm lint:***
```

### Production

```
$ pnpm build
```

### Preview

```
$ pnpm preview
```
