## Usage

1. Install dependencies

```bash
$ yarn
```

2. Create a page

```bash
$ yarn create:page pageA
```

3. Replace the `main.js` path in `index.html`

```html
<script type="module" src="/src/pages/pageA/main.js"></script>
```
